Piano Js is Customizable virtual keyboard written in pure JavaScript. Piano js is browser friendly, common JS and Es6 friendly so can just import it, rquire it, or use script tag to use it. Piano js comes with differents layout you can check it here, or build your own !
You can find all the doc on the git repo: https://github.com/soixantecircuits/piano
js-standard-style Join the chat at https://gitter.im/soixantecircuits/piano

Default layout

data-piano-layout="azerty"
data-piano-scale="1"

Email layout

data-piano-layout="azerty-mail"
data-piano-scale="1"

Right top positionned default layout

data-piano-layout="azerty"
data-piano-scale="1.5"
data-piano-position="right, top"

Center middle positionned azerty layout

data-piano-layout="azerty"
data-piano-event-id="input-2"
data-piano-position="center, middle"

Absolute 150 - 150 positionned azerty layout

data-piano-layout="azerty"
data-piano-animation-in="bounceInUp"
data-piano-animation-out="fadeOutUp"
data-piano-position="absolute, 150, 20"
